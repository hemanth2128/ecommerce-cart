# E-Commerce Cart System

A responsive E-Commerce Cart System built using React.js, Zustand, and Zod. This project demonstrates global state management and form validation while providing users with a seamless shopping cart experience.

## Features

* Product listing with images and pricing
* Add items to cart
* Increase / decrease quantity
* Remove products from cart
* Dynamic total price calculation
* Global state management using Zustand
* Checkout form validation using Zod
* Responsive UI design
* Professional product card layout

## Technologies Used

* React JS
* Zustand
* Zod
* JavaScript (ES6+)
* CSS3
* Vite

## Installation

1. Clone repository

```bash
git clone <repository-link>
```

2. Install dependencies

```bash
npm install
```

3. Run project

```bash
npm run dev
```
